# 画像テキストエディタ

Vue 3、TypeScript、Viteを使用した画像テキスト編集アプリケーションです。このアプリケーションでは、画像をアップロードし、テキストを追加して編集することができます。

## 主な機能

- **画像アップロード**: ファイル選択またはドラッグ＆ドロップで画像をアップロード
- **テキスト編集**: 画像上に直接テキストを追加・編集
- **テキストスタイル設定**: フォント、サイズ、色、太字/斜体などのスタイル設定
- **テキスト効果設定**: 回転、影、アウトラインなどの効果設定
- **編集した画像のダウンロード**: PNG形式で保存可能
- **PWA対応**: オフライン使用可能、ホーム画面に追加可能
- **レスポンシブデザイン**: モバイルデバイスにも対応

## 技術スタック

- **フロントエンド**: Vue 3 + TypeScript
- **ビルドツール**: Vite
- **キャンバス操作**: Fabric.js
- **状態管理**: Pinia
- **PWA**: Vite PWA Plugin

## アプリケーションの構成

アプリケーションは以下のコンポーネントで構成されています：

1. **FabricCanvas**: Fabric.jsキャンバスを表示し、テキスト操作を行うコンポーネント
2. **TextStylePanel**: テキストのスタイル（フォント、サイズ、色など）を設定するパネル
3. **TextEffectsPanel**: テキストの効果（回転、影、アウトラインなど）を設定するパネル
4. **ImageUploader**: 画像のアップロード（ファイル選択、ドラッグ＆ドロップ）を行うコンポーネント
5. **InstallPrompt**: PWAのインストールを促すプロンプト

また、以下のコンポーザブルを使用して機能を実装しています：

1. **useFabricCanvas**: Fabric.jsキャンバスの操作を行うコンポーザブル
2. **useFabricText**: テキストの操作を行うコンポーザブル
3. **useFileHandling**: ファイル操作（画像のアップロード、ダウンロードなど）を行うコンポーザブル
4. **usePwa**: PWA関連の機能を実装するコンポーザブル

## インストール

```bash
# リポジトリのクローン
git clone https://github.com/yourusername/image-editor.git
cd image-editor

# 依存関係のインストール
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:5173/ にアクセスすると、アプリケーションが表示されます。

## ビルド

```bash
npm run build
```

ビルドされたファイルは `dist` ディレクトリに出力されます。

## 使い方

1. 「画像を選択」ボタンをクリックするか、画像をドラッグ＆ドロップしてアップロード
2. 「T」ボタンをクリックしてテキストを追加
3. テキストをダブルクリックして直接編集
4. サイドバーのパネルでテキストのスタイルや効果を設定
5. 「保存」ボタンをクリックして編集した画像をダウンロード
